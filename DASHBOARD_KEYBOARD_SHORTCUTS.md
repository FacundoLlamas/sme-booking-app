## Detailed Guide

### Navigation Shortcuts

#### Go to Bookings
- **Keys:** Cmd+B (Mac) or Ctrl+B (Windows/Linux)
- **Use:** Quickly navigate to the Bookings page to manage all bookings
- **Tip:** Use Cmd+N to create a new booking after navigating here

#### Go to Calendar
- **Keys:** Cmd+C (Mac) or Ctrl+C (Windows/Linux)
- **Use:** Jump to the Calendar view to see bookings visually
- **Tip:** Use arrow keys to navigate between dates

#### Go to Analytics
- **Keys:** Cmd+A (Mac) or Ctrl+A (Windows/Linux)
- **Use:** Access analytics and reports dashboard
- **Tip:** Use arrow keys to navigate through different views

#### Go to Settings
- **Keys:** Cmd+S (Mac) or Ctrl+S (Windows/Linux)
- **Use:** Access business settings and preferences
- **Warning:** Save your changes before leaving this page

#### Go to Customers
- **Keys:** Cmd+U (Mac) or Ctrl+U (Windows/Linux)
- **Use:** Navigate to the customer management page
- **Tip:** Use Cmd+F to search for customers

#### Go to Home
- **Keys:** Cmd+H (Mac) or Ctrl+H (Windows/Linux)
- **Use:** Return to the dashboard home page
- **Tip:** All your recent bookings and metrics are here

### Booking Shortcuts

#### Create New Booking
- **Keys:** Cmd+N (Mac) or Ctrl+N (Windows/Linux)
- **Use:** Open the new booking creation dialog
- **Requires:** Navigation to Bookings page first (Cmd+B)
- **Tip:** After this shortcut, use Tab to navigate form fields

#### Export Bookings
- **Keys:** Cmd+E (Mac) or Ctrl+E (Windows/Linux)
- **Use:** Export the current bookings to CSV or JSON
- **Format:** Choose CSV for Excel, JSON for data integration
- **Location:** Works on Bookings and Calendar pages

#### Refresh Data
- **Keys:** Cmd+R (Mac) or Ctrl+R (Windows/Linux)
- **Use:** Reload all data from the server
- **Use Case:** When data seems outdated or after batch operations
- **Tip:** You can also press F5 to refresh the entire page

### Quick Action Shortcuts

#### Search
- **Keys:** Cmd+K (Mac) or Ctrl+K (Windows/Linux)
- **Use:** Open the global search bar
- **Features:**
  - Search bookings by customer name
  - Search by service type
  - Filter by date range
- **Tip:** Type to filter immediately

#### Help & Shortcuts
- **Keys:** Cmd+/ (Mac) or Ctrl+/ (Windows/Linux)
- **Use:** Open help documentation
- **Content:** Includes FAQs and troubleshooting

#### Keyboard Shortcuts List
- **Keys:** Cmd+? (Mac) or Ctrl+? (Windows/Linux), or just ?
- **Use:** Display this complete keyboard shortcuts reference
- **Format:** Organized by category (Navigation, Actions, etc.)
- **Interactive:** Click on any shortcut to learn more

### Text Editing in Forms

#### Undo
- **Keys:** Cmd+Z (Mac) or Ctrl+Z (Windows/Linux)
- **Use:** Undo your last action
- **Limit:** Typically supports 10-20 undo steps

#### Redo
- **Keys:** Cmd+Shift+Z (Mac) or Ctrl+Shift+Z (Windows/Linux)
- **Use:** Redo an action after undo
- **Tip:** Can be used multiple times in succession

#### Cut / Copy / Paste
- **Cut:** Cmd+X (Mac) or Ctrl+X (Windows/Linux)
- **Copy:** Cmd+C (Mac) or Ctrl+C (Windows/Linux)
- **Paste:** Cmd+V (Mac) or Ctrl+V (Windows/Linux)
- **Use:** Standard clipboard operations for text fields
- **Tip:** Copy booking details to share with team

#### Select All
- **Keys:** Cmd+A (Mac) or Ctrl+A (Windows/Linux)
- **Use:** Select all text in a form field
- **Tip:** Use before Copy to grab all form data

### Navigation & Dialog Interactions

#### Close Dialog
- **Keys:** Escape
- **Use:** Close any open modal, dialog, or popup
- **Behavior:** Applies to all overlays
- **Tip:** Works from anywhere in the dialog

#### Confirm Action
- **Keys:** Enter / Return
- **Use:** Submit forms or confirm actions
- **Location:** Works on focused buttons and form fields
- **Tip:** After filling a form, just press Enter

#### Tab Navigation
- **Tab:** Move to next element
- **Shift+Tab:** Move to previous element
- **Use:** Navigate through form fields or menu items
- **Tip:** Faster than clicking for keyboard users

#### Numeric Navigation (1-6)
- **Keys:** 1, 2, 3, 4, 5, 6
- **Use:** Jump to specific sections or items
- **Example:**
  - Press 1 to go to the first booking
  - Press 2 to go to the second booking
- **Tip:** Great for quickly accessing recent bookings

## Tips & Tricks

### Power User Tips
1. **Combine shortcuts:** Use Cmd+B (go to Bookings) + Cmd+N (new booking) to quickly create
2. **Search first:** Use Cmd+K to search before navigating manually
3. **Keyboard only:** Navigate entirely without a mouse using arrow keys + Enter
4. **Mobile:** Long-press the Help button (?) to see shortcuts on mobile

### Accessibility
- All shortcuts respect system preferences (dark mode, reduced motion)
- Works with screen readers and voice control
- No conflicting shortcuts with assistive technologies
- Touch-friendly on tablets and large screens

### Common Workflow

**Creating a new booking for a customer:**
1. Press Cmd+K (search)
2. Type customer name
3. Press Escape to close search
4. Press Cmd+B (go to Bookings)
5. Press Cmd+N (new booking)
6. Use Tab to fill form fields
7. Press Enter to submit

**Reviewing and exporting bookings:**
1. Press Cmd+B (go to Bookings)
2. Press Cmd+R (refresh data)
3. Press Cmd+E (export)
4. Select CSV format
5. File downloads to Downloads folder

## Customization

Some shortcuts can be customized in Settings:
- Go to Cmd+S (Settings)
- Navigate to "Keyboard Shortcuts"
- Modify any shortcut to your preference
- Changes sync across all devices

## Troubleshooting

**Shortcuts not working?**
1. Check if you're in a text input field (most shortcuts disabled there)
2. Try pressing Escape first to ensure you're not in a dialog
3. Check browser compatibility above
4. Try the alternative (Cmd on Mac, Ctrl on Windows)

**Browser conflicts?**
- If your browser uses the same shortcut, it will take precedence
- Try using the dialog help (?) to see which shortcuts are available
- Check browser settings to disable conflicting shortcuts
